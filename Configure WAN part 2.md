# Часть 2. Настройка PPP с аутентификацией CHAP

## Шаг 1. Настройка последовательных интерфейсов

Для обеспечения безопасного соединения между R3 и R1973 используется протокол PPP с аутентификацией CHAP.

**Параметры аутентификации:**
- Имя пользователя на R3: `R1973`
- Имя пользователя на R1973: `R3`

# Заключение

В ходе выполнения практической работы были настроены:

1. Базовая конфигурация маршрутизаторов с назначением IP-адресов
2. PPP-соединение с аутентификацией CHAP между R3 и R1973
3. Динамическая маршрутизация OSPFv2 для IPv4
4. BGP между AS 3 и AS 1973
5. DHCP-ретранслятор на R1 для получения IP-адресов PC0
6. IPv6-адресация на всех маршрутизаторах
7. OSPFv3 для IPv6
8. EIGRPv6 между R3 и R1973

Все протоколы работают корректно, связность между устройствами обеспечена.
